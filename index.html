jQuery Plugin: Convert Table Column Numbers

This plugin allows you to convert numbers a column in an html table. You can also chain the table with this plugin to resolve conversion methods. Divide the argument of the jQuery function with ":" for the first being the conversion used, and the second indicating the column you need to convert. Examples: 
            $("table").convertNumber("toPercent:2");
        
            $("table").convertNumber("toDecimal-2:2");
        
            $("table").convertNumber("lengthConvert-meterTOcentimeter:2").convertNumber("toDecimal-2:2");
        
            $("table").convertNumber("toDecimal-2:2").convertNumber("toCurrency-dollar:2");
